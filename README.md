# Job Portal

A simple job portal web application connecting job seekers and employers.

## Features

asd

- **Job Listings** - Browse and view all available job postings
- **User Authentication** - Login and register for candidates and employers
- **Job Search** - Find jobs that match your profile
- **Employer Details** - View detailed company information
- **Contact** - Get in touch with the job portal team
- **About Us** - Learn more about the platformasd

## Pages

- `home.html` - Landing page
- `login.html` - User login
- `register.html` - New user registration
- `view_all_jobs.html` - Browse all job listings
- `employerdetails.html` - Employer/company details
- `candidates.html` - Candidate profiles
- `companies.html` - Company listings
- `aboutus.html` - About the platform
- `contact_us.html` - Contact information

## Getting Started

1. Open `home.html` in your web browser
2. Explore job listings or create an account
3. Use the navigation menu to access different sections

## Files

- `script.js` - Main JavaScript functionality
- `style.css`, `styles1.css`, `styles2.css` - Styling
- `vercel.json` - Deployment configuration
